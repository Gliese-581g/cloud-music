# cloud-music

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
## 项目描述
一个用Vue全家桶开发的仿网易云音乐单页应用

### 功能实现
1. 顶栏组件
2. 搜索详情页
3. 推荐页：
   * banner
   * 歌单  
4. 排行榜页
5. 歌手页
6. 播放页
7. 底部播放控件
